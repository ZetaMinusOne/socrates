**Step 2: Install the plugin**
```

```
Expected: plugin installs cleanly with no errors. No submodule init, no build step, no manual configuration. If it fails with "plugin not found," verify that marketplace.json `plugins[0].name` is `socrates-skill` and `source` is `./socrates`.

**Step 5: Test session hook**
```
/clear
/socrates Is the Socratic method still relevant to modern education?
```
Expected: works without manual SKILL.md read (session hook injects context). Note: if the hook does not fire (known bug #16538 for plugin-based hooks), this is a known limitation — document it but do not fail the E2E gate on this alone.

**Pass criteria:** Steps 1-4 all succeed. Step 5 success is preferred but not blocking (known upstream bug).
